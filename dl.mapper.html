class dl.mapper.ibase.IBaseMapper¶

Bases: abc.ABC

Base interface for all data mappers

You will see parameter ‘obj’ that is not typed. The type of object is dependent on the child class. This interface should not be used, as it only serves as a common ancestor for other interfaces.
